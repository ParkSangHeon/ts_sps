# standard-project-structure

## 목적
개발에서 사용하는 표준 프로젝트(라이브러리, 폴더구조 및 구성)를 생성하여 공유하고 좀더 손쉽게 표준 프로젝트를 이용해 개발을 시작할수 있도록 하고 사용하는 라이브러리와 폴더구성등을 공통적으로 사용하는 목적에 있다.

## 필수항목
* 폴더구조 정리
* 라이브러리 정리
* 간단한 테스트 및 설명(README.md)

## 표준화 리스트를 추가하는 방법
    // orphan branch 로 생성
    checkout --orphan NEW_BRANCH_NAME

## 표준화 리스트
* typescript nodejs
